# Umbra Eternal

Un RPG por turnos Roguelike con estética Detailed Pixel Art ambientado en un mundo de High Fantasy oscura siendo devorado por El Vacío.

## Lore y Atmósfera

El mundo de Umbra Eternal está siendo devorado por El Vacío. Es un entorno de High Fantasy oscura donde no existe la benevolencia pura:

- **Ángeles**: Heraldos del Orden Absoluto. Fríos y prefieren el exterminio total y la purificación por fuego.
- **Demonios**: Buscan el caos y contratos ventajosos, viendo al Vacío como un competidor.
- **El Vacío**: Fuerza entrópica que consume la realidad, mutando nodos, enemigos y héroes.

## Sistemas del Juego

### 1. Sistema de Héroes (Composición Bi-Clase)
- **Módulo Especie (70% Stats Base)**: Define HP, ATK, DEF, SPD, EVA, CRIT + Mutador Racial
- **Módulo Clase (30% Stats + Habilidades)**: Define crecimiento de estadísticas y catálogo de habilidades

### 2. Sistema de Combate
- Combate 5v5 por turnos con iniciativa dinámica
- Formación de pirámide: 2 protectores adelante, 3 detrás
- Sistema de cobertura táctica
- Jerarquía de daño: Físico, Mágico, Primordial

### 3. Módulo de Run (The Void Deck System)
- Draft de cartas de tres mazos (A, B, C)
- Corrupción por exposición de las cartas no elegidas
- Nodos: Combate, Santuario, Tienda, Evento, Pacto

### 4. Economía y Meta-progresión
- Pactos con facciones (Ángeles, Demonios, Vacío)
- Archivo Eterno para desbloqueos permanentes
