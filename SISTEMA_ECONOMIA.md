# Economía y Meta-progresión

## Definición Base

Sistema económico y de progresión persistente que conecta las runs individuales con el desbloqueo permanente de contenido, creando un ciclo de mejora a largo plazo.

## Economía de Run

### Moneda Principal: Oro
- **Obtención**: Combates, eventos, venta de equipo
- **Usos**: Comprar equipo, mejoras, servicios en tiendas
- **Escalado**: Enemigos más difíciles dan más oro

### Economía de Pactos
- **Coste de Sacrificio**: Recursos permanentes vs beneficios temporales
- **Tipos de Sacrificio**:
  - **HP Máximo**: Reducción permanente
  - **Habilidades**: Bloqueo temporal o permanente
  - **Equipamiento**: Pérdida de items aleatorios
  - **Humanidad**: Afecta endings y desbloqueos

### Sistema de Raridad de Equipamiento
- **Común (Blanco)**: Stats básicos, sin afijos especiales
- **Infrecuente (Verde)**: 1-2 afijos menores
- **Raro (Azul)**: 2-3 afijos, uno puede ser especial
- **Épico (Púrpura)**: 3-4 afijos, habilidades especiales
- **Legendario (Naranja)**: 4-5 afijos, efectos únicos de set
- **Corrupto (Morado/Negro)**: Stats elevados pero con penalizaciones

### Sistema de Afijos
#### Afijos Ofensivos
- **Daño Elemental**: +X% daño de un tipo específico
- **Penetración**: Ignora Y% de defensa/enemigos
- **Velocidad de Ataque**: +Z% SPD en combate
- **Crit Amplificado**: +W% daño de golpes críticos

#### Afijos Defensivos
- **Resistencia Elemental**: -X% daño de un tipo
- **Absorción**: Recupera Y HP cada Z turnos
- **Evasión Mejorada**: +W% EVA vs ataques específicos
- **Bloqueo**: Chance de bloquear completamente daño

#### Afijos de Utilidad
- **Cooldown Reducido**: -X% cooldown de habilidades tipo Y
- **Economía**: +Y% oro/comida
- **Experiencia**: +Z% experiencia ganada
- **Movimiento**: +W% velocidad en mapa

## Sistema de Pactos

### Facciones y Filosofía

#### Pactos Angelicales (Orden Absoluto)
- **Oferta**: Poder purificador, equipo sagrado, curación masiva
- **Coste**: Restricciones comportamentales, pérdida de libertad
- **Ejemplo**: "Bendición de Luz" (+50% daño vs seres oscuros) a cambio de "No puedes usar habilidades demoníacas"

#### Pactos Demoníacos (Caos y Contratos)
- **Oferta**: Poder corrupto, mutations, control sobre enemigos
- **Coste**: Corrupción progresiva, pérdida de humanity
- **Ejemplo**: "Firma Infernal" (+30% todos los stats) a cambio de "Pierdes 10% HP máximo cada combate"

#### Pactos del Vacío (Entropía y Mutación)
- **Offer**: Poder transformador, acceso a mechanics prohibidas
- **Coste**: Mutaciones permanentes, inestabilidad creciente
- **Ejemplo**: "Ojo del Vacío" (Ves enemies ocultos, +25% crit) a cambio de "20% de mutar aleatoriamente cada 3 combates"

### Sistema de Contratos
- **Duración**: Temporal vs Permanent
- **Coste**: Inmediato vs Diferido
- **Anulación**: Posible con gran sacrificio o item específico
- **Stacking**: Cómo interactúan múltiples pactos

## Archivo Eterno (Meta-progresión)

### Sistema de Desbloqueo Permanente

#### Especies Desbloqueables
- **Humanos**: Inicialmente disponibles
- **Elfos Oscuros**: Completar 3 runs con builds basados en sigilo
- **Demonios Menores**: Sobrevivir a 5 pactos demoníacos en una run
- **Ángeles Caídos**: Rechazar 3 pactos angelicales consecutivamente
- **Ser del Vacío**: Alcanzar corrupción máxima y sobrevivir

#### Clases Desbloqueables
- **Guerrero**: Disponible inicialmente
- **Mago Arcano**: Desbloquear al alcanzar nivel 5 con cualquier personaje
- **Pícaro**: Completar run sin ser detectado por enemies élite
- **Paladín**: Forjar alianza Ángel-Demonio
- **Nigromante**: Domar 3 enemies élite en una misma run

#### Variante de Habilidades
- **Mejoras Raciales**: +10% stats base para especies desbloqueadas
- **Variantes de Habilidades**: Versiones alternativas con diferentes efectos
- **Synergies Ocultas**: Combinaciones especies-clase con bonus únicos

### Sistema de Currency Meta

#### Esencia del Vacío
- **Obtención**: Enemies corruptos, pactos con Vacío, achievements específicos
- **Usos**: Desbloquear contenido premium, acelerar progresión
- **Raridad**: Común, Infrecuente, Rara, Mítica

#### Fragmentos de Memoria
- **Obtención**: Completar runs, descubrir lore
- **Usos**: Desbloquear narrativas, endings alternativos
- **Colección**: Completar sets para bonus permanentes

### Sistema de Mejora Permanente

#### Mejoras Generales
- **Stats Base**: +2% HP/ATK/DEF para todos los héroes
- **Economía**: +5% oro por run completada
- **Experiencia**: +10% XP ganada
- **Lucky**: +1% chance de items raros

#### Mejoras Específicas
- **Por Especie**: Bonificaciones específicas para cada especie
- **Por Clase**: Mejoras a habilidades base de cada clase
- **Por Tipo de Run**: Mejoras que se activan bajo ciertas condiciones

## Sistema de Progression

### Niveles de Archivo Eterno
- **Nivel 1-10**: Desbloqueos básicos
- **Nivel 11-25**: Contenido intermedio
- **Nivel 26-50**: Mechanics avanzadas
- **Nivel 51+**: Endgame y challenges

### Milestones y Achievements
- **Primera Run Completada**: Desbloquea segunda especie
- **10 Runs Completadas**: Desbloquea mazo B extendido
- **Primer Jefe Derrotado**: Desbloquea tienda extendida
- **Corrupción Máxima**: Desbloquea ending del Vacío

### Sistema de Challenges
- **Diarios**: Objetivos rotativos con recompensas
- **Semanales**: Challenges más complejos
- **Mensuales**: Desafíos cooperativos o comunitarios
- **Permanentes**: Challenges legendarios con rewards únicos

## Balance y Escalado

### Sistema de Catch-up
- **Bonus para nuevos jugadores**: Accelerated progression inicial
- **Mentorías**: Bonificaciones por ayudar a jugadores nuevos
- **Events especiales**: Oportunidades de acelerar desbloqueos

### Sistema de Economy Balance
- **Inflation control**: Ajuste dinámico de costs/rewards
- **Sink economy**: Métodos para remover exceso de currency
- **Value rebalancing**: Ajustes periódicos para mantener economy saludable

## Integración con Otros Sistemas

### Con Sistema de Héroes
- Mejoras permanentes afectan a todos los héroes
- Variante de habilidades dependen de desbloqueos
- Synergies especies-clase mejorables

### Con Sistema de Combate
- Nuevos tipos de daño desbloqueables
- Mechanics avanzadas de combate
- Enemies con abilities únicas

### Con Sistema de Run
- Nuevas cartas para el Void Deck
- Tipos de nodos adicionales
- Events especiales desbloqueables
